# Insurance Data Analysis Project

## 📊 Overview
This project performs exploratory data analysis (EDA) on medical insurance data to understand how factors like age, BMI, smoking status, and region affect insurance charges.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📁 Dataset
The dataset contains information about individuals including:
- Age
- Sex
- BMI
- Number of children
- Smoking status
- Region
- Insurance charges

## 📈 Key Insights
- Smokers have significantly higher insurance charges.
- Age shows a positive relationship with insurance cost.
- BMI has a moderate impact on charges.
- Insurance cost varies by lifestyle factors.

## 📊 Visualizations
The project includes:
- Distribution of insurance charges
- BMI vs Charges scatter plot
- Smoker vs Non-smoker comparison
- Correlation heatmap

## 🚀 How to Run
1. Clone the repository
2. Install required libraries:pip install pandas matplotlib seaborn
3. Open `insurance_analysis.ipynb` in Jupyter Notebook
4. Run all cells
